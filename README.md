本日の学習進捗（2026/03/20）
今日は、以前学習したデータ処理の核となる NumPy と Pandas の知識を改めて整理し、基礎の復習と実践的な定着を図りました。
学習内容：
- NumPy入門: 
  - ndarray オブジェクトの属性（shape, ndim, size）の理解。
  - 多様な配列生成関数（arange, zeros, ones, linspace等）の習得。
  - 数値計算における基本関数（ceil, floor, abs, where等）の活用。

- Pandas基礎:
  - Series および DataFrame オブジェクトの構造と作成方法。
  - 辞書やリストからのデータ変換。
  - loc（ラベル指定）と iloc（行番号指定）によるデータ抽出の使い分け。
  - 基本的な統計メソッド（value_counts, describe, sort_values）の利用。
  
- DataFrame基礎:
  - DataFrameの基礎: 属性（ndim, shape, size）の理解と基本操作。
  - データ抽出: Boolean Indexingを用いた条件フィルタリング。
  - 統計解析: describe()やinfo()を用いた探索的データ解析（EDA）の基礎。
  - 入出力処理: CSV, TSV, Excelなど多様なファイル形式のハンドリング。


本日の学習進捗（2026/03/21）
本日は、データ分析の基盤となる「データ操作」から、実務で不可欠な「データの結合」までを体系的に学習しました。
  - 1. データ処理の基礎
  - 多次元配列 (NumPy): ndarray の属性（shape, dtype）の理解と、効率的なベクトル計算の実装。
  - データ構造 (Pandas): DataFrame と Series の基本操作、要約統計量（平均、合計、カウント等）によるデータ概況の把握。

  - 2. データの結合・統合
  - pd.concat(): 行方向および列方向へのデータの単純連結。
  - pd.merge(): SQLライクな結合（Inner, Left, Right, Outer）の実践。
  - 実データ演習: SQLite3 を使用し、データベースからのデータ読み込みと、複数テーブルを紐付けた集計処理の自動化。
